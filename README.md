## Daily Updates

GitHub Actions refreshes the hub every day at **UTC 00:00 / Beijing 08:00**.
The pipeline searches recent papers, updates datasets, fetches teaser images, enriches
OpenAlex citation metadata, backfills TLDR/deep analysis fields, rebuilds the Astro site,
and publishes GitHub Pages.

Required repository secret: `ARK_API_KEY`.
Optional secrets: `ARK_API_BASE_URL`, `ARK_MODEL_NAME`, `SMART_MODEL_NAME`,
`MINERU_API_KEY`, `OPENALEX_API_KEY`, `OPENALEX_MAILTO`.
