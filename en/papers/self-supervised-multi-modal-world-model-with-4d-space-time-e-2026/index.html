Deep Analysis

Innovations

  • Earth4D: a novel planetary-scale 4D space-time positional encoder extending 3D multi-resolution hash encoding to include time, enabling sub-meter, sub-second precision across centuries
  • Self-supervised multi-modal world model fusing vision-language encoders with Earth4D embeddings via masked reconstruction
  • Learnable hash probing that surpasses a multi-modal foundation model pre-trained on substantially more data

Methodology

DeepEarth uses Earth4D, a 4D space-time positional encoder that extends 3D multi-resolution hash encoding to incorporate time, allowing efficient planetary-scale coverage. Multi-modal encoders (e.g., vision-language models) are fused with Earth4D embeddings and the entire model is trained via masked reconstruction.

Key Results

Earth4D achieves state-of-the-art performance on an ecological forecasting benchmark. With learnable hash probing, it surpasses a multi-modal foundation model pre-trained on substantially more data.
