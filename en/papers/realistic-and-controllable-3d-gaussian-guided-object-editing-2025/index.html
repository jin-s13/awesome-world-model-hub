Deep Analysis

Innovations

  • Using 3D Gaussian representation as dense prior injected into denoising process for accurate pose control and spatial consistency
  • Scene-level 3D bounding box layout to reconstruct occluded areas of non-target objects
  • Hierarchical fine-grained features as additional conditions for appearance details
  • Unified framework supporting object repositioning, insertion, and deletion

Methodology

G^2Editor leverages a 3D Gaussian representation of the edited object as a dense prior, injected into the denoising process to ensure accurate pose control and spatial consistency. A scene-level 3D bounding box layout is employed to reconstruct occluded areas of non-target objects, and hierarchical fine-grained features are incorporated as additional conditions to guide appearance details during generation. The method is evaluated on the Waymo Open Dataset.

Key Results

Experiments on the Waymo Open Dataset demonstrate that G^2Editor effectively supports object repositioning, insertion, and deletion within a unified framework, outperforming existing methods in both pose controllability and visual quality, while also benefiting downstream data-driven tasks.
