Deep Analysis

Innovations

  • Episodic memory mechanism for video-world-model policies to address long-horizon non-Markovian manipulation
  • Recap-Cue (RC) module with Perceiver-based Recap Compressor that maps per-frame CLIP embeddings into compact memory tokens
  • Cue Gate that estimates task completion from memory and language
  • Delta-reconstruction auxiliary loss and episode-boundary supervision for training memory without per-frame progress labels
  • Unified memory injection applicable to both UNet and Diffusion Transformer (DiT) backbones via cross-attention

Methodology

MemoryVAM integrates an episodic memory module into video-world-model policies. The Recap-Cue module uses a Perceiver-based Recap Compressor to encode per-frame CLIP embeddings into compact memory tokens, and a lightweight Cue Gate estimates task completion from memory and language. These tokens are injected into both the video backbone and action decoder via cross-attention, conditioning actions on history. The model is trained with video prediction, a delta-reconstruction auxiliary loss, and episode-boundary supervision, requiring no per-frame progress labels.

Key Results

On LIBERO-Mem, MemoryVAM improves average success from 5% to 42.5%. On real robots, it achieves 78.3% success on counting tasks, 80.0% on spatial recall, and 75.0% on sequential tracking.
