PredRNN: A Recurrent Neural Network for Spatiotemporal Predictive Learning

TLDR

PredRNN introduces a recurrent network with decoupled memory cells and zigzag memory flow for spatiotemporal predictive learning, achieving competitive results on five datasets.

Reasoning

The paper presents a novel architecture with explicit memory decoupling and cross-layer communication, supported by ablation studies and strong empirical results on multiple benchmarks.
