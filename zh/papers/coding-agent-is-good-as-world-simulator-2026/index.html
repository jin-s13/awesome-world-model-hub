深度分析

创新点

  • 通过可执行仿真代码构建基于物理的世界模型的智能体框架，结合了规划、代码生成、视觉审查和物理分析智能体。
  • 基于视觉和物理反馈对仿真代码进行迭代修正，以强制执行物理约束并匹配提示要求。
  • 在物理准确性、指令忠实度和视觉质量上优于基于视频的世界模型。

方法

该框架协调四个智能体：规划智能体将自然语言提示转换为结构化场景计划，代码智能体将计划实现为可执行仿真代码，视觉审查智能体提供视觉反馈，物理分析智能体检查物理一致性。根据反馈迭代修正代码，直到仿真符合提示要求和物理约束。

关键结果

所提出的框架在物理准确性、指令忠实度和视觉质量上优于先进的基于视频的模型，并可应用于驾驶仿真和具身机器人任务。
